Resumo

Formado em Sistema para Internet no Instituto Brasileiro de Tecnologia Avançada (IBTA), trabalho com desenvolvimento Frontend desde 2014.

Nos primeiros contatos que tive com desenvolvimento de softwares, a criação de interfaces e interações sempre me fascinou. A forma como pequenos detalhes em um layout ou em um fluxo de interações podem mudar significativamente a forma como aquela interface vai ajudar o usuário me faz querer me aprofundar cada vez mais nessa área.

Procuro criar layout limpos e claros, sempre pensando em um código legível e semântico. Afinal, programamos não apenas para que a máquina entenda, mas para que outros seres humanos possam compreender o código.

Sempre acompanhando blogs de desenvolvimento Frontend, UX, UI, pegando dicas e seguindo tendências nesse mundo que se renova a cada dia. Gosto de seguir pessoas ativas na comunidade, elas geralmente tem um excelente conteúdo a ser repassado.

Para cada projeto novo que inicio, procuro agregar algo novo. Seja no código que escrevo, na forma como lido com o projeto, ou até mesmo como me comunicar com a minha própria equipe. Tenho em mente que cada projeto deve ser uma “versão melhorada” do projeto anterior.
